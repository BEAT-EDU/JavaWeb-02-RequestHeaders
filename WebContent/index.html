Using Request Headers

Reminder: the source code from the online tutorial runs only on Tomcat 7 or other servers that support servlets 3.0 and later. This project is an alternative version that has been modified to run on Tomcat 6 and other servers that support servlets 2.4 or 2.5. The main changes here from the code in the online tutorials is that I removed the @WebServlet annotation (and associated import statement) and added a web.xml file that gave url-patterns to the servlets.
